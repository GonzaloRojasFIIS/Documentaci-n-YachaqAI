# Tema 2.2: Ethernet y Capa de Enlace

La capa de enlace de datos agrupa bits en tramas, maneja direcciones físicas (direcciones MAC) y detecta errores de transmisión.

- **[[2. conceptos/ethernet.md|Ethernet]]**: Es el protocolo LAN por excelencia definido bajo el estándar IEEE 802.3.
- **[[2. conceptos/switch.md|Switch Capa 2]]**: Dispositivo de red que lee direcciones MAC de destino en las tramas Ethernet para reenviarlas únicamente por el puerto adecuado.
