# Tema 3.1: Protocolo IP y Enrutamiento

La capa de red se encarga del direccionamiento lógico y el envío de paquetes a través de diferentes redes interconectadas.

- **[[2. conceptos/ipv4.md|Protocolo IPv4]]**: Define direcciones de 32 bits representadas en formato decimal con puntos.
- **[[2. conceptos/routing.md|Routing (Enrutamiento)]]**: Es el proceso mediante el cual los routers determinan la ruta óptima para un paquete utilizando tablas de routing.
