# Cuestionario: Subnetting

## Pregunta 1

¿Qué información se obtiene de una dirección IP y su máscara?

> Respuesta: El ID de red, la dirección de broadcast y el rango de hosts válidos.

## Pregunta 2

¿Por qué subnettear una red?

> Respuesta: Para reducir broadcast, optimizar direcciones y mejorar seguridad.

## Pregunta 3

¿Cuántos hosts útiles tiene una subred /28?

> Respuesta: 14 hosts utilizables.
