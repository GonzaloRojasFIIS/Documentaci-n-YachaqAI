# Cuestionario: Routing

## Pregunta 1

¿Cuál es la diferencia entre routing estático y dinámico?

> Respuesta: El estático se configura manualmente; el dinámico aprende rutas mediante protocolos.

## Pregunta 2

¿Qué dispositivo realiza routing típicamente?

> Respuesta: Un router o un switch capa 3.

## Pregunta 3

¿Qué es una tabla de routing?

> Respuesta: Una base de datos que almacena las rutas conocidas hacia redes destino.
