# Cuestionario: Uso de Herramientas (Tool Calling)

## Pregunta 1

¿Qué es la llamada a herramientas o Tool-calling en agentes?

> Respuesta: Es la capacidad técnica que permite a un LLM decidir autónomamente invocar APIs, bases de datos o scripts externos para realizar acciones físicas o recopilar información del exterior.
