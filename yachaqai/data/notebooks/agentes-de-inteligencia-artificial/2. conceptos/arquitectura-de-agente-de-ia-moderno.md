# Arquitectura de Agente de IA Moderno

## Definición
Estructura técnica de un agente de inteligencia artificial impulsado por LLMs que integra varios módulos para permitir la autonomía. Incluye un LLM como cerebro, memoria, planificación/razonamiento, herramientas/APIs, y módulos de percepción y acción.

## Ideas clave
- LLM como componente central o "cerebro"
- Memoria a corto y largo plazo para mantener contexto
- Módulo de planificación para dividir tareas complejas
- Integración de herramientas/APIs para interactuar con el entorno
- Módulos de percepción y acción para el mundo real
