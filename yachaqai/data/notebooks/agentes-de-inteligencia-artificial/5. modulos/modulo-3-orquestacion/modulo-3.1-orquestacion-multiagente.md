# Tema 3.1: Orquestación de Agentes y Multiagentes

La [[2. conceptos/orquestacion-de-agentes.md|Orquestación de Agentes]] coordina múltiples inteligencias autónomas especializadas que colaboran entre sí para resolver problemas de gran escala mediante flujos conversacionales estructurados o grafos dirigidos.

## Frameworks Principales

- **LangGraph**: Permite diseñar flujos con bucles, ciclos y bifurcaciones utilizando la representación de grafos dirigidos (DAGs).
- **CrewAI**: Modela sistemas de agentes con roles definidos (ej: Investigador, Escritor) asignados a tareas específicas.
- **AutoGen**: Habilita conversaciones libres y fluidas entre múltiples agentes para resolver problemas cooperativamente.

Mediante estos sistemas, un agente puede delegar tareas específicas a otros sub-agentes, optimizando la latencia y la calidad de la respuesta en flujos complejos.
